# 당장(Dangjang)

전통시장의 마감 할인 상품을 한눈에 확인할 수 있도록 도와주는 웹 서비스입니다.  
사용자는 시장과 가게, 상품, 할인 정보를 쉽게 조회할 수 있고, 관리자는 시장/가게/상품/할인 정보를 등록 및 관리할 수 있습니다.

## 프로젝트 소개

전통시장은 다양한 먹거리와 상품을 저렴하게 제공하지만, 젊은 층과 1인 가구에게는 어떤 상품이 있는지, 지금 어떤 할인 상품이 나와 있는지 알기 어렵다는 문제가 있습니다.  
특히 시장에서는 마감 시간대에 할인 판매가 자주 이루어지지만, 이 정보가 온라인으로 정리되어 있지 않아 소비자가 접근하기 어렵고 상인도 추가 매출 기회를 충분히 얻지 못하는 경우가 많습니다.

**당장**은 이런 문제를 해결하기 위해 만들어진 서비스입니다.  
사용자는 현재 시장에서 판매 중인 상품과 마감 할인 상품을 쉽게 확인할 수 있고, 상점 및 관리자 측에서는 할인 상품을 등록하여 더 많은 고객 유입을 기대할 수 있습니다.

## 기획 배경

- 전통시장 정보의 디지털 접근성이 낮음
- 젊은 층과 1인 가구는 시장 상품 정보를 미리 파악하기 어려움
- 마감 할인 상품 정보를 온라인으로 접할 수 있게 하면 퇴근길 또는 배달을 시키는 상황에서 합리적인 가격으로 원하는 음식을 먹을 수 있음
- 1인가구나 젊은 층이 음식 포장을 위해 시장을 방문을 하면 시장의 다른가게도 방문을 할 가능성이 생기고, 시장 전체에 매출이 오를 것으로 기대됨

이 서비스는 전통시장 상품 정보를 온라인으로 구조화하고, 특히 **마감 할인 상품**에 집중하여 시장 활성화를 목표로 합니다.

## 핵심 목표

- 사용자가 전통시장 내 할인 상품을 쉽게 확인할 수 있도록 한다.
- 전통시장과 상점 정보를 디지털화하여 접근성을 높인다.
- 마감 할인 정보를 실시간에 가깝게 제공하여 소비자 유입을 유도한다.
- 전통시장 상인의 판매 기회를 확대하고 재고 소진을 돕는다.

## 주요 기능

### 사용자 기능
- 시장 목록 조회
- 시장 상세 조회
- 가게 목록 조회
- 가게 상세 조회
- 상품 목록 조회
- 상품 상세 조회
- 할인 상품 목록 조회
- 할인 상품 상세 조회
- 시장명/가게명/상품명 기반 검색

### 관리자 기능 (MVP 단계)
- 시장 등록/수정
- 가게 등록/수정
- 상품 등록/수정
- 할인 상품 등록/수정

> MVP 단계에서는 별도의 로그인/회원 시스템 없이 `X-Admin-Key` 헤더 기반으로 관리자 권한을 처리합니다.

## MVP 범위

현재 프로젝트는 핵심 기능만 포함한 MVP 형태로 개발 중입니다.

### 포함 기능
- 시장/가게/상품/할인 정보 등록 및 조회
- 할인 상품 목록 제공
- 검색 기능
- 관리자 전용 등록/수정 기능
- 공통 응답 및 공통 예외 처리

### 이후 추가 기능
- 회원가입 / 로그인
- 장바구니 / 결제
- 예약 / 픽업 주문
- 알림 기능
- 리뷰 / 즐겨찾기
- 통계 대시보드
